# Распознавание дорожных знаков с помощью YOLOv8

Автор: Габараев Тимур Альбертович  
Группа: 33ИС  

## Описание работы
В рамках проекта разработана модель для автоматического поиска и классификации дорожных знаков на видео и изображениях. В качестве основы использовалась архитектура YOLOv8s. Модель адаптирована под работу в реальном времени и обучена на выборке из 6000 кадров.

## Стек технологий
Python 3.10, PyTorch, Ultralytics YOLOv8, OpenCV, Matplotlib, среда разработки Google Colab (графический процессор T4).

## Датасет
Использован готовый набор данных с разметкой под YOLO. Все знаки разделены на 6 основных классов: знаки ограничения, приоритета, предписывающие знаки, «Стоп», «Въезд запрещен» и знаки поворота. Соотношение выборок: 80% на обучение, 20% на валидацию и тесты.

## Результаты
* Максимальная точность (метрика mAP@0.5) составила 0.812 после 50 эпох обучения.
* Точность выросла на 6.9% по сравнению со стандартной базовой моделью.
* Скорость обработки кадра на видео составляет около 12 миллисекунд, что позволяет использовать систему в реальном времени.


## Инструкция по запуску
1. Открыть ноутбук из папки notebooks в Google Colab.
2. В настройках среды выполнения выбрать графический процессор (T4 GPU).
3. Запустить выполнение всех ячеек. Датасет скачается и настроится автоматически.
